CMake 2.8.0
  • FindCVS:

    The module defines the following variables:

       CVS_EXECUTABLE - path to cvs command line client
       CVS_FOUND - true if the command line client was found

    Example usage:

       find_package(CVS)
       if(CVS_FOUND)
         message("CVS found: ${CVS_EXECUTABLE}")
       endif(CVS_FOUND)
