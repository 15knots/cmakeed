CMake 2.8.6
  • FindGit:

    The module defines the following variables:

       GIT_EXECUTABLE - path to git command line client
       GIT_FOUND - true if the command line client was found

    Example usage:

       find_package(Git)
       if(GIT_FOUND)
         message("git found: ${GIT_EXECUTABLE}")
       endif()
